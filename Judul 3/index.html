Sering Ditanyakan

Masih ragu? Temukan jawaban Anda di sini.

Apakah saya butuh alat khusus seperti komputer kasir?
Tidak perlu! SIBBO dirancang untuk berjalan lancar cukup menggunakan Smartphone Android atau Tablet Anda. Hemat biaya investasi alat.
Bisakah digunakan saat tidak ada internet (offline)?
Saat ini SIBBO membutuhkan koneksi internet untuk menyimpan data secara real-time agar aman dan bisa diakses dari mana saja. Namun, penggunaan datanya sangat hemat.
Apakah data penjualan saya aman?
Sangat aman. Kami menggunakan enkripsi standar industri untuk melindungi data Anda. Hanya Anda yang bisa mengakses laporan keuangan usaha Anda.
