Mini E-Library

2025 — Backend Developer

Mini E-Library App is a web application designed for managing and accessing digital book metadata, built with Laravel and React integrated via Inertia.js. Administrators can perform CRUD operations on books, including uploading covers and PDF files, manage categories, and monitor usage through a dashboard with statistics and charts. Registered users can easily search, filter, view book details, read or download available PDFs, add titles to their favorites, and submit ratings and reviews. The application features user authentication with email verification (using Mailpit locally) and offers personalized dashboards displaying reading history and favorited books.
